Registrarse en ngrok, en este caso con mi mail institucional.
De forma gratuita, ngrok permite un solo punto de conección con nombre por defecto.
Se puede usar cualquier otro punto de conección, puente, túnel o lo que cumpla la función.

Usando ngrok. Primero se establece la autenticación, que se guardará en un archivo local
(el valor a insertar se consigue desde la página de ngrok, logueado a la cuenta):
    ngrok config add-authtoken XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
    devuelve:   Authtoken saved to configuration file: C:\Users\User\AppData\Local/ngrok/ngrok.yml
    

El script que toma los datos anteriores desde google, no puede hacer magia, por lo que necesita un link hacia el archivo.
A su vez, el archivo tiene que ser compartido de forma pública para obtener ese link.

Para compartir la hoja de cálcuo:
    archivo,
    compartir a internet,
    seleccionar el número de hoja correcto,
    en CSV.
Así se obtendrá el link para introducir en el script "cargarDB - archivo desde google.py"
    

Luego, continuando en la misma hoja de google, es necesario configurar un "desencadenante" para que cuando haya una nueva inserción desde el formulario asociado a la hoja, se envíen esos nuevos datos al archivo "server.py".
Para eso vamos a:
    Extensiones,
    Apps Script.

En Apps Script seleccionamos "Crear un nuevo proyecto", donde se pegará el siguiente código:

function onFormSubmit(e) {
  const sheet = e.source.getActiveSheet();
  const row = e.range.getRow();
  const data = sheet.getRange(row, 1, 1, sheet.getLastColumn()).getValues()[0];

  const headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0];
  const rowData = {};
  headers.forEach((header, i) => {
    rowData[header.trim()] = data[i];
  });

  // Enviar al backend Python (tu servidor Flask)
  const url = "https://nonenlightening-liana-unresiliently.ngrok-free.dev/api/nuevo_proyecto";
  const options = {
    method: "post",
    contentType: "application/json",
    payload: JSON.stringify(rowData)
  };
  UrlFetchApp.fetch(url, options);
}
